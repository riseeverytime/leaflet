Examples

Different examples are available in the example folder to present various use cases:

  • A simple Marker with Popup
  • Event handling
  • Viewport usage
  • Vector layers (Rectangle, Circle, etc.)
  • Other layers (LayerGroup, FeatureGroup)
  • Tooltips
  • Custom zoom control
  • Layers control
  • Custom panes
  • Draggable marker toggle
  • Map with bounds
  • Custom component: a list of markers
  • Animated map
  • WMS tile layer
  • Video overlay

These examples can be run locally by cloning the repository, installing the dependencies and running npm run examples or yarn run examples if you're using Yarn.
